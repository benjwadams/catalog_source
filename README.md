Catalog Source
============
Source code and configuration files for the GLOS Catalog

Uses environmental variables:

    BASEX_SERVER (no default)
    BASEX_PORT (1984)
    BASEX_USER (no default)
    BASEX_PASS (no default)
  
Use basex client
./basexclient -n 64.9.200.113 -p 1984 -P glos -U user
